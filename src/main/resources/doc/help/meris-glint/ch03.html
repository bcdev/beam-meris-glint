3.1. Input Products

The MERIS GLINT processor requires as input:

  • MERIS L1b product (mandatory for both FLINT and AGC parts)
  • AATSR L1b product (mandatory if a Glint value from the FLINT processor shall be used in the GGAC algorithm, otherwise not needed).

The FLINT computation is performed for the overlapping areas of the MERIS/AATSR scenes. For the areas covered by MERIS only, just the AGC computation is provided.
