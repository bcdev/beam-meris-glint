3.2. Output Products

3.2.1. Standard Quantities

The GLINT output product provides the following standard quantities:

  • tau550 (Spectral aerosol optical depth at 550nm)
  • tau778 (Spectral aerosol optical depth at 778nm)
  • tau865 (Spectral aerosol optical depth at 865nm)
  • btsm (Total supended matter scattering)
  • atot (Absorption at 443 nm of all water constituents)
  • ang443/865 (Aerosol Angstrom coefficient)

3.2.2. Optional Quantities

The GLINT output product provides the following optional quantities (as defined by the user):

  • TOSA Reflectances (MERIS bands 1-10, 12, 13)
  • Water Leaving Reflectances (MERIS bands 1-10, 12, 13)
  • Path Reflectances (MERIS bands 1-10, 12, 13)
  • Downwelling Irradiance Transmittances (MERIS bands 1-10, 12, 13)

The output product can be saved in standard BEAM DIMAP format.
