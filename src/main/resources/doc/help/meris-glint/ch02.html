2.1. The MERIS Glint Processor as BEAM plugin

As a BEAM plugin, the MERIS Glint processor needs the following software to be installed in advance:

  • BEAM, version 4.9 or higher

The BEAM software can be obtained from the BEAM download page (www.brockmann-consult.de/beam).

The GLINT software delivery consists of five jar files: the AGC and FLINT main modules and adjusted BRR (Bottom of Rayleigh Reflectance), SDR (Surface Directional Reflectance) modules and L2Auxdata, which provide additional Java classes used by GLINT.

  • flint-processor-1.1.2.jar
  • beam-meris-glint-1.2.jar
  • beam-meris-brr-2.3.jar
  • beam-meris-sdr-2.3.jar
  • beam-meris-l2auxdata-1.2.jar

These files need to be copied into the 'modules' folder of the BEAM installation (Figure 2.1). Note that existing versions of these jar files need to be replaced/overwritten. There should not be more than one jar file for each of these modules in the 'modules' folder!
